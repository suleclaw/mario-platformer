# Mario Platformer

A Mario-style 2D platformer built with Phaser 3. No external image or audio files — everything is procedurally drawn and generated.

**Live:** https://mario-platformer.vercel.app

## Features
- 2 scrolling levels with increasing difficulty
- Procedurally drawn sprites, platforms, and decorations
- Touch D-pad controls (left, right, jump) for mobile/tablet
- Keyboard support (Arrow keys / WASD + Space)
- Stomping enemies, coin collection, lives system
- Procedural 8-bit sound effects
- Parallax scrolling backgrounds
- Particle effects on coin collection
- Start, Level Complete, Game Over, and Win screens
